Mis Proyectos

Buzón Tributario del Estado de Durango v.3

Coordinación integral del proyecto desde la planeación estratégica hasta el despliegue en la red y servidores del Gobierno del Estado de Durango. Aplicación de metodologías ágiles como Scrum y Kanban, así como enfoques híbridos y Waterfall según la etapa del proyecto. Gestión de equipos multidisciplinarios, cronogramas y entregables clave.

ERP / CRM para PyMEs

Gestión del ciclo completo de desarrollo e implementación de un sistema ERP/CRM personalizado para pequeñas y medianas empresas. Recolección de requerimientos, diseño funcional, desarrollo modular y capacitación a usuarios finales. Integración de ventas, facturación, inventarios y atención al cliente.

Formalización de Departamento de TI

Diseño y ejecución de un plan para la creación y consolidación del área de Tecnologías de la Información. Incluyó la definición de estructura operativa, implementación de procesos, selección de herramientas, reclutamiento de personal técnico y capacitación en buenas prácticas y metodologías ágiles.

Buzón Tributario del Estado de Durango v.2

Estabilización del sistema en su segunda versión, abarcando la mejora del código, pruebas funcionales y automatizadas, despliegue en infraestructura interna del Estado de Durango y liberación segura para su uso por parte de contribuyentes. Coordinación con áreas jurídicas, fiscales y técnicas.

Sistema de Control de Llantas

Planificación, diseño y desarrollo de un sistema web para la gestión de llantas y mantenimiento preventivo de flotillas. Automatización de registros, alertas de desgaste y control de inventario. Interfaz amigable y adaptable para usuarios administrativos y operativos.

API consulta de RFC nacional OCSP - SAT

Gestión de Desarrollo de API que consulta el OCSP de Servicio de Administración Tributaria consultando la validez de los certificados de la e.firma.

App de Control para Sistema de Riego

Diseño y desarrollo de aplicación móvil en React Native con integración vía Bluetooth a dispositivos ESP32. Permite controlar y programar horarios de riego de forma remota. Proyecto orientado a optimizar el uso de recursos hídricos en entornos agrícolas pequeños y medianos.

Página Web Corporativa para RECUVEA

Diseño y desarrollo de sitio web institucional para RECUVEA, empresa dedicada a soluciones de papelería, tecnología y equipo de cómputo. Implementación responsive, enfoque en posicionamiento SEO básico y estructura para mostrar catálogo de productos y contacto comercial.
